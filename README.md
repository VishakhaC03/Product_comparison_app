# Product Comparison App

A modern and interactive Product Comparison Web Application built using React JS, Vite, Tailwind CSS, HTML, CSS, JavaScript, and Axios.

The application allows users to search, filter, compare products side-by-side, and select the best product after comparison.

---

## Features

- Product Search Functionality
- Category-Based Product Filtering
- Compare Multiple Products Side-by-Side
- Dynamic Product Selection & Removal
- Product Details Page
- Choose Best Product after Comparison
- Duplicate Product Prevention
- Category Validation for Relevant Comparison
- Smart Alert Messages
- Responsive UI Design
- Shared Navbar Across Pages
- API Integration using Axios
- React Router Navigation

---

## Key Functionalities

### Product Search
Users can search products dynamically using a live search feature.

### Category Filtering
Products can be filtered by category to improve browsing experience.

### Product Comparison
Users can compare products side-by-side and view feature differences.

### Best Product Selection
After comparison, users can select one product as the best choice and view its details.

### Smart Validation
- Prevents duplicate product comparison
- Restricts irrelevant category comparison
- Displays alerts for invalid actions
